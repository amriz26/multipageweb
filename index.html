How to Apply CSS to HTML

To include an external CSS file in an HTML document, use the <link> element in the <head> section: 

<link rel="stylesheet" href="styles.css">

How to Use JavaScript in a Webpage

To link to an external JavaScript file, use the <script> element in the HTML file: 

<script src="script.js"></script>
To write JavaScript directly in the HTML file, place code within the <script> tags: 
<script>
    console.log("Hello, world!");
</script>
